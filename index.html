About Impulse

Impulse web player is a JavaScript-based YouTube music player.

The app works in the browser with no server-side code.
The music data is read from a JSON file and displayed. The data is managed on my local system and cannot be edited in the browser in any way.

I created Impulse to have my own music library instead of it being on a 3rd party service.

You can contact me (the developer) via email: markus.vaananen(at)gmail.com
